text formatting examples

bold, italics, underline, strikethru, unordered and ordered lists

text formatting : strong, em, u and strike tags

Use a strong tag to bold text in html

Use an em, or i, tag to put a word in italics

Use a u tag to underline a word but that is discouraged - gets confused as a link

Use a strike tag to strikethrough a word

text formatting : unordered lists

Use a ul tag for unordered lists, li tag for line items
